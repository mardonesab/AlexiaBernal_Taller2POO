Pokémon - Camino para ser el Mejor

Proyecto de Programación Orientada a Objetos (Java)

Es un simulador sencillo de Pokémon que hice para el taller 2.

Lo que tiene el juego:
- Nueva partida o continuar partida
- Capturar Pokémon en diferentes zonas
- Revisar tu equipo
- Cambiar Pokémon en el PC
- Retar gimnasios uno por uno
- Desafiar al Alto Mando
- Curar Pokémon
- Guardar la partida

Cómo ejecutarlo:
1. Ejecutar la clase Main.java
2. Los archivos .txt deben estar en la carpeta datos

Archivos principales:
- Main.java
- Pokemon.java
- Jugador.java
- GestorJuego.java
- TablaTipos.java
- Gimnasio.java
- MiembroAltoMando.java
